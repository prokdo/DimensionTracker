# DimensionTracker

[🇷🇺 Русский](README_RU.md) | [🇬🇧 English](README.md)

Плагин для Minecraft серверов, основанных на Paper, который окрашивает имена игроков в чате и TAB-списке в зависимости от текущего измерения

## Возможности

- Цвет имени в чате в зависимости от измерения
- Цвет имени в TAB-списке в зависимости от измерения
- Не требует настройки, работает из коробки

## Цвета

| Измерение               | Цвет       |
| ----------------------- | ---------- |
| Верхний мир (Overworld) | Зелёный    |
| Нижний мир (Nether)     | Красный    |
| Край (The End)          | Фиолетовый |

## Требования

- Версия Minecraft: 1.21+
- Ядро сервера: Paper
- Версия Java: 21+

## Установка

1. Скачать актуальный `.jar` файл со страницы [Releases](https://github.com/prokdo/DimensionTracker/releases)
2. Поместить файл в папку `plugins` сервера
3. Перезапустить сервер

> **Примечание:** Плагин не требует `/reload` — рекомендуется полная перезагрузка сервера

## Сборка из исходников

> **Примечание:** Требуются JDK 21 или выше и Maven

```bash
# Клонировать репозиторий
git clone https://github.com/prokdo/DimensionTracker.git
cd DimensionTracker

# Собрать с помощью Maven
mvn clean package

# Собранный jar будет лежать в target/DimensionTracker-<version>.jar
```

## Решение проблем

| Проблема                           | Решение                                                                                           |
| ---------------------------------- | ------------------------------------------------------------------------------------------------- |
| Плагин не загружается              | Проверьте консоль сервера на ошибки при запуске                                                   |
| Имя не окрашивается в чате или TAB | Убедитесь, что используете Paper 1.21+ и другие плагины / дата-паки не переопределяют отображение |
| Всё ещё есть проблемы?             | Откройте [Issue](https://github.com/prokdo/DimensionTracker/issues) с подробностями               |
