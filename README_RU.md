Плагин для Minecraft серверов, основанных на Paper, который окрашивает имена игроков в чате и TAB-списке в зависимости от текущего измерения

## Возможности

- Цвет имени в чате в зависимости от измерения
- Цвет имени в TAB-списке в зависимости от измерения
- Полная настройка цветов для каждого измерения и мира (поддерживаются HEX и именованные цвета)
- Поддержка кастомных миров
- Возможность независимо отключить окрашивание в чате или TAB
- Встроенное определение AFK с настраиваемым таймаутом и цветом
- Ручное переключение AFK
- Не требует начальной настройки, работает из коробки

## Цвета

| Измерение               | Цвет       |
| ----------------------- | ---------- |
| Верхний мир (Overworld) | Зелёный    |
| Нижний мир (Nether)     | Красный    |
| Край (The End)          | Фиолетовый |
| AFK                     | Серый      |

## Требования

- Версия Minecraft: 1.21+
- Ядро сервера: Paper
- Версия Java: 21+

## Установка

1. Скачать актуальный `.jar` файл со страницы [Releases](https://github.com/prokdo/DimensionTracker/releases)
2. Поместить файл в папку `plugins` сервера
3. Перезапустить сервер

## Конфигурация

Файл конфигурации находится по пути `plugins/DimensionTracker/config.yml` и создаётся автоматически при первом запуске со следующим содержимым:

```yaml
chat:
  enabled: true

tab:
  enabled: true

colors:
  overworld: "#00FF00" # или GREEN / green
  nether: "#FF0000" # или RED / red
  end: "#FF55FF" # или LIGHT_PURPLE / light_purple
  default: "#FFFFFF" # или WHITE / white


  # Пример кастомного мира:
  # имя_кастомного_мира: "#RRGGBB" или стандартное имя цвета
```

> **Примечание:** Плагин поддерживает обновление конфигурации "на лету" через `/dimensiontracker reload` или `/dt reload` — полная перезагрузка сервера не требуется

Цвета можно указывать как HEX-значения (`#RRGGBB`) или именованными цветами (`green`, `red`, `white` и т.д.)

Для кастомных миров добавьте запись с именем папки мира в качестве ключа. Имена кастомных миров имеют приоритет над цветами по измерению. Любой мир, не указанный в конфигурации, использует цвет `default`

## Сборка из исходников

> **Примечание:** Требуются JDK 21 или выше и Maven

```bash
# Клонировать репозиторий
git clone https://github.com/prokdo/DimensionTracker.git
cd DimensionTracker

# Собрать с помощью Maven
mvn clean package

# Собранный jar будет лежать в target/DimensionTracker-<version>.jar
```

## Решение проблем

| Проблема                              | Решение                                                                                                                                              |
| ------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| Плагин не загружается                 | Проверьте консоль сервера на ошибки при запуске                                                                                                      |
| Имя не окрашивается в чате или TAB    | Убедитесь, что используете Paper 1.21+ и другие плагины / дата-паки не переопределяют отображение, а соответствующие функции включены в конфигурации |
| Изменения конфигурации не применяются | Выполните `/dt reload` или перезапустите сервер                                                                                                      |
| Цвет кастомного мира не работает      | Убедитесь, что ключ совпадает с именем папки мира точь-в-точь                                                                                        |
| Всё ещё есть проблемы?                | Откройте [Issue](https://github.com/prokdo/DimensionTracker/issues) с подробностями                                                                  |
